Angularjs Datepicker

A simple pure angularjs directive
that generates a responsive datepicker around your input
Choosen date is {{date1}} ...

Usage

Html

Wrap your input into the datepicker element

Javascript

Add the datepicker module dependency
angular.module('app', [ '720kb.datepicker' ]);

Options

Date format

All the accepted date formats can be found here
date-format="{{pattern}}"

Customize buttons

Just a FontAwesome buttons example button-prev="" button-next=""
